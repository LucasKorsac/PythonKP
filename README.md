Func.py — генерация простых чисел
	Содержит класс Generator.
		Основные методы:
		MillerRabin(n, k) — вероятностный тест простоты числа.
		GenPr(bit) — генерирует обычное простое число длиной bit бит.
		GenStrongPr(bit) — генерирует «сильное» простое число (p = 2q1q2 + 1).

Класс используется при создании ключей RSA для p и q.
Принцип работы: случайно генерируется число, проверяется тестом Миллера-Рабина, пока не найдется подходящее простое.

RSA.py — генерация ключей RSA
	Класс RSA создаёт пару ключей:
		p и q — простые числа (обычные, близкие или сильные).
		n = p * q — модуль RSA.
		e — открытая экспонента (по умолчанию 65537).
		d — закрытая экспонента, обратная к e по модулю φ(n).
		Опция c=True делает p и q близкими (уязвимо для метода Ферма).
		Опция st=True использует сильные простые.
Принцип работы: генерируются простые числа → вычисляется модуль → выбирается e → вычисляется d.


Factor.py — факторизация
	Класс Factorization для разложения n:
		Fermat(n) — метод Ферма, эффективен если p и q близки.
		TrialDiv(n) — метод полного перебора, находит делитель n.
Принцип работы: либо приближение a² - b² для близких чисел (Ферма), либо простая проверка делителей (TrialDiv).

TimingAttack.py — моделирование атаки по времени
	Класс TimingAttack:
		VulnerPow(base, exp, mod) — уязвимое возведение в степень по битам, время зависит от значений битов ключа.
		MeasureTime(base, exp, mod) — измеряет время выполнения VulnerPow.
Принцип работы: имитация атаки на RSA, где время операции зависит от секретного ключа.

Exp.py — проведение эксперимента
	Класс Experiment с методом run(samp, bit, c, st):
		Генерирует несколько ключей RSA.
		Измеряет время факторизации методом Ферма.
		Собирает данные в таблицу pandas.DataFrame.
Принцип работы: циклично генерирует ключи → факторизует n → записывает время → возвращает таблицу с результатами.

Graph.py — визуализация результатов
	Функции для построения графиков:
		PlotRes(df) — среднее время факторизации в зависимости от размера ключа.
		PlotExperiments(df) — отображает все отдельные эксперименты.
		PlotWithTheory(df) — сравнение с теоретической кривой (не рабочий)
		PlotTimingAttack(res) — график атаки по времени для одного ключа.
Принцип работы: использует matplotlib для визуализации, группируя результаты по параметрам (обычные/сильные/близкие простые).

Interf.py — графический интерфейс (Tkinter)
	Класс Gui:
		Поля для ввода размера ключа и количества экспериментов.
		Кнопки: генерация ключа, запуск эксперимента, построение графиков.
		Вывод параметров ключа и результатов экспериментов.
Класс использует потоки для запуска эксперимента без зависания интерфейса.
Принцип работы: пользователь вводит параметры → генерирует ключ → запускает эксперимент → смотрит графики и таблицы.
